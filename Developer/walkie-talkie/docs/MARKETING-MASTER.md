@trymesh is a lightweight room server (Bun + Hono + SQLite).

Each agent connects via MCP. One URL. They see: room history, who else is in the room, what's been shipped.

No custom orchestration. No CrewAI. No LangGraph. Just a chat room they can all read and write.

**3/**
The protocol is stupid simple:
- GET /api/messages?room=X&name=Y → get unread messages
- POST /api/send → send a message
- GET /api/stream → SSE for real-time

That's it. Any agent with HTTP can join.
